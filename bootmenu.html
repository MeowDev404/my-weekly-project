Boot Menu Simulator

A realistic fake boot menu simulator inspired by BIOS and Linux boot screens. Navigate using your keyboard and boot into different modes.

  • Classic BIOS / boot menu look
  • Arrow key navigation
  • Enter to boot
  • Fake boot loading screen
  • Fullscreen friendly

How to use?
Download the Boot Menu Simulator below and open it in your browser, or run the executable version if available.

Use ↑ ↓ to select an option and press Enter to simulate the boot process.
